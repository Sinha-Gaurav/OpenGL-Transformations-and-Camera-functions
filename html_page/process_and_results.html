Objective
Understand openGL transformations, matrix stack operations, and 3D viewing pipeline.
Loading Models
We used the assimp library to import models stored as .obj files to OpenGL. Three models were made using blender and four models were downloaded from free3d.com.
The seven models are:-
  1. Car
  2. Table
  3. Chair
  4. Potted plant
  5. Teapot with cups
  6. Mario character
  7. Platform
Using Transformations
We used opengl transformations to relocate the models and create a scene of our choice. Each model was given a color(which was sent to the fragment shader through a uniform) and a transformation matrix which positions the model into its appropriate place in the scene. Once the transformation matrices were computed for each model, they are used inside the render loop before each draw call of the model.
Screenshots of different angles of the scene
Camera Functions
We used the glm library to implement camera functions to move around in our scene using keyboard and mouse as inputs.
There are 3 main types of camera functions implemented.
  1. Pan: Pan helps in moving the camera to different positions and we used the keys W, A, S, D, Q, E for forward, backward, left, right, up and down respectively.
  2. Yaw, Pitch and Roll: These are the 3 rotations about v, u and n directions respectively.
  3. Zoom: This changes the distance of the center of projection to the front plane aka fov.
